HTML, CSS, JAVASCRIPT

Html : è il linguaggio di markup standard utilizzato per creare pagine web. Viene utilizzato per strutturare il contenuto delle pagine web, come testi, immagini, link e altri elementi. HTML non è un linguaggio di programmazione, ma un linguaggio di markup, il che significa che non esegue operazioni, ma definisce la struttura e la semantica dei contenuti.
Css : è un linguaggio utilizzato per definire lo stile e l'aspetto di un documento HTML. Mentre l'HTML si occupa della struttura e del contenuto della pagina web, il CSS controlla come questi contenuti vengono visualizzati nel browser, come colori, font, spaziatura, layout e altre caratteristiche estetiche.
JavaScipt : è un linguaggio di programmazione utilizzato per rendere le pagine web interattive e dinamiche. A differenza di HTML (che definisce la struttura della pagina) e CSS (che definisce lo stile e l'aspetto), JavaScript è usato per aggiungere comportamenti e funzionalità interattive alla pagina web.
